## 用法 
1. 新增，删除，编辑 item
2. 可根据时间 文本匹配搜索历史记录，时间为范围
3. 总体展示店名，是否为like，位置
4. 同步交互，是否同步完成指示
5. item细节 
    1. 时间默认为新增时间，可修改
    2. 店名 总价 总评分 文本评价 单店面可添加为like
    3. 位置，gps定位 or 文字描述（地铁优先，，跳转美团？）
    4. 添加 各项菜名 菜评分 菜文本评价 单菜品可添加为like 
## 技术细节
